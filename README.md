# TO TEST DE PROJECT #  
## 1. Start JSON server ## 
-open a terminal go to /src/api/ then run   
`json-server --port 3001 --watch data.json` 
## 2. Run dev server ##   
-on another terminal, from root folder run  
`npm run dev`  
-if you get message Next is not an internal command run  
 `npm i next`
